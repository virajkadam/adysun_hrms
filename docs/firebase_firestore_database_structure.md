## üîó Collection Relationships & Connections

### Primary Relationships:

1. **Admin Authentication Flow**:
   ```
   admins ‚Üí admin_sessions
   ```
   - `admins.id` ‚Üí `admin_sessions.adminId`
   - Used for session management and authentication

2. **Employee Management Flow**:
   ```
   employees ‚Üí employments ‚Üí salaries
   ```
   - `employees.id` ‚Üí `employments.employeeId`
   - `employments.id` ‚Üí `salaries.employmentId`
   - `employees.id` ‚Üí `salaries.employeeId`

3. **Leave Management Flow**:
   ```
   employees ‚Üí leaves
   ```
   - `employees.id` ‚Üí `leaves.employeeId`

4. **Audit Trail Connections**:
   ```
   admins ‚Üí (createdBy/updatedBy) ‚Üí all collections
   ```
   - `admins.id` ‚Üí `employees.createdBy/updatedBy`
   - `admins.id` ‚Üí `employments.createdBy/updatedBy`
   - `admins.id` ‚Üí `salaries.createdBy/updatedBy`
   - `admins.id` ‚Üí `leaves.approvedBy`
